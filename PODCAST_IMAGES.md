# Podcast Package Images

To complete the podcast setup page, you need to add three images to the `/public/images/` directory:

## Required Images:

1. **`podcast-basic.jpg`** - Basic podcast setup image
   - Should show: Simple microphone setup, basic audio interface
   - Recommended size: 400x250 pixels
   - Format: JPG or PNG

2. **`podcast-pro.jpg`** - Professional podcast setup image  
   - Should show: Multi-microphone setup, mixing board, professional studio
   - Recommended size: 400x250 pixels
   - Format: JPG or PNG

3. **`podcast-enterprise.jpg`** - Enterprise podcast setup image
   - Should show: Complete professional studio, multiple cameras, broadcast equipment
   - Recommended size: 400x250 pixels  
   - Format: JPG or PNG

## How to Add Images:

1. Save your images in the `public/images/` folder
2. Name them exactly as listed above
3. The page will automatically display them
